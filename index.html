ABOUT OUR PROJECT

🌐 Building Solutions Through Teamwork

This website was developed as part of the COS 121 (Introduction to Computer Programming) course at the University of Uyo, Akwa Ibom State, Nigeria. The project is a collaborative effort involving students from different departments and faculties, demonstrating teamwork, creativity, and practical programming skills in the design and development of a modern web application.

💡 Purpose of the Project

This project was created as part of the requirements for the COS 121 – Introduction to Computer Programming course at the University of Uyo. It provides students with an opportunity to apply the programming concepts learned in the classroom to the development of a real-world web application.



The project encourages collaboration among students from different departments and faculties, allowing them to work together, share ideas, and combine their skills to achieve a common goal. It also helps students develop problem-solving, communication, and project management skills that are essential in the technology industry.



Through the design and implementation of this website, students gain practical experience in planning, developing, testing, and documenting software while following professional standards and best practices in web development.

🎯 Project Goal

To design and develop a professional, user-friendly, interactive components, mobile compatibility and responsive website.
